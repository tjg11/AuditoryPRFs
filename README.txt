Collecton of Python scripts for finding Auditory Population Receptive Fields (PRFs) based on MRI trial data. All three scripts are activley being updated and changed, and may not be entirely functional. General updates and functionality on each script below:

stimulus_creation.py: converted to function format, can handle multiple files, expects rigid directory structure

convolve_hrf.py: not in function format, can only handle one trial run currently, working on updating to functionality of stimulus_creation

find_prf.py: most in progress script, activley learning how to find PRFs
